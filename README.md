Comparação entre Regressão Logística e Boosting na Predição de Inadimplência
Descrição

Este repositório contém o código desenvolvido para o Trabalho de Conclusão de Curso (TCC), cujo objetivo é comparar o desempenho de modelos de machine learning na predição de inadimplência.

Modelos utilizados
Regressão Logística
LightGBM
XGBoost
Base de dados

Dataset: Home Credit Default Risk (Kaggle)

Metodologia

O pipeline inclui:

Tratamento de dados faltantes
Engenharia de atributos
Codificação de variáveis categóricas
Treinamento dos modelos
Avaliação com métricas AUC e índice de Gini
Como executar
Instalar dependências:
pip install -r requirements.txt
Executar o script:
python script_analise_risco_credito.py
Resultados

Os modelos foram comparados com base na curva ROC e índice de Gini.
